## Architecture
- AWS의 Route53, ALB를 통해 서브도메인에 따른 트래픽을 제어하고, HTTPS를 적용시켰습니다.
- Github Action, S3, Code Deploy를 통해 Github Push, Megre에 따른 CI, CD 프로세스를 적용했습니다.
